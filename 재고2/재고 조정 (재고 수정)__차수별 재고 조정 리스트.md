# 페이지 분석 — 차수별 재고 조정 리스트

## 1. 페이지 개요

| 항목 | 내용 |
|------|------|
| 메뉴 그룹/위치 | 재고 조정 (재고 수정) |
| 페이지명 | 차수별 재고 조정 리스트 |
| 식별 코드 | 미기재 |
| 목적 | 재고조정(R-STK-606)에서 처리된 조정 이력을 일자/차수 단위로 집계·관리하는 이력 조회 메뉴. 정기 감사(Audit) 및 내부 보고용으로 활용 |
| 접근 권한 | 물류사 관리자(조회+다운로드), 시스템 관리자(전체) |

> **차수(次數)란?** 재고조정 작업의 실행 회차를 의미한다. 예: 1차 조정(2024-03-01), 2차 조정(2024-03-15) 등. 동일 날짜에 여러 차수의 조정이 발생할 수 있다.

---

## 2. 업무 흐름 (Business Flow)

```
[재고조정 승인 완료] (R-STK-606)
    ↓
시스템이 해당 조정을 '차수' 단위로 자동 그룹핑하여 이력 저장
    ↓
[차수별 재고 조정 리스트] 에서 일자/차수별 조정 이력 조회
    ↓
특정 차수 클릭 → 해당 차수에 포함된 재고별 조정 상세 내역 조회
    ↓
엑셀 다운로드 → 내부 보고 / 화주사 정산 / 감사 자료 활용
```

## 7. 예외 처리 및 알림

| 상황 | 처리 방식 |
|------|----------|
| 조회 기간 범위 과다 (1년 초과) | "조회 기간은 최대 1년(365일)까지 설정할 수 있습니다." 경고 |
| 조회 결과 없음 | "조회 조건에 해당하는 재고조정 이력이 없습니다." 안내 |
| 대용량 엑셀 다운로드 (10,000건 초과) | 비동기 처리 후 완료 시 다운로드 링크 알림 |
| 특정 차수 데이터 무결성 오류 | 해당 차수에 경고 아이콘 표시 및 시스템 관리자 알림 |
