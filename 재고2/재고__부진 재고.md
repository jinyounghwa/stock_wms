# 페이지 분석 — 부진 재고

## 1. 페이지 개요

| 항목 | 내용 |
|------|------|
| 메뉴 그룹/위치 | 재고 |
| 페이지명 | 부진 재고 |
| 식별 코드 | 미기재 |
| 목적 | 오랜 기간 입출고 이동이 없거나 유효기간이 임박한 재고(부진재고)를 식별·시각화하여 처리 방향(할인 판매, 폐기, 위치 재배치 등)을 결정할 수 있도록 지원하는 분석 대시보드 메뉴 |
| 접근 권한 | 화주사(자사 재고만), 물류사 관리자, 시스템 관리자 |

> **패션 WMS의 부진재고**: 패션 산업은 시즌성이 강해 SS(봄여름)·FW(가을겨울) 시즌 간 전환 시 전시즌 재고가 대량으로 부진재고화된다. 보관일수·마지막 출고일을 기준으로 90일/180일/365일 단위로 부진 등급을 나누고, 화주사·물류사가 공동으로 처리 방향을 결정하는 프로세스가 일반적이다.

---

## 2. 업무 흐름 (Business Flow)

```
[재고 보관 지속] (입출고 없이 장기 보관)
    ↓
시스템 배치 (일 1회) → 부진재고 등급 자동 갱신
(최종 출고일 기준 90일/180일/365일 초과 여부 판정)
    ↓
[부진 재고] 대시보드 에서 현황 확인
    ↓
시각화 차트로 규모·분포 파악
    ↓
상세 목록에서 처리 대상 선별
    ↓
처리 방향 결정
  ├── 할인 판매 → 화주사에게 통보
  ├── 폐기 → 재고조정 요청 (R-STK-606, 감소)
  ├── 위치 재배치 → 창고 내 이동 처리
  └── 반품 → 화주사 반송 출고 처리
```

### 3-3. 시각화 차트 (대시보드 영역)

**① 부진 등급별 재고 분포 (도넛 차트)**
- 정상 / 90일~180일 / 180일~365일 / 365일 이상 비율 시각화

**② 브랜드·카테고리별 부진재고 현황 (막대 차트)**
- X축: 브랜드 또는 카테고리
- Y축: 부진재고 수량
- 색상: 부진 등급 구분

**③ 보관일수 분포 히스토그램**
- X축: 보관일수 구간 (0~30일, 31~60일, 61~90일, 91~180일, 181~365일, 365일 초과)
- Y축: 재고 건수

**④ 로케이션별 부진재고 적치 현황 (히트맵 또는 테이블)**
- 창고 내 어느 존·로케이션에 부진재고가 집중되어 있는지 시각화
- 공간 활용 최적화 판단에 활용

**⑤ 적치이력 타임라인 (상세 팝업)**
- 선택 재고의 로케이션 이동 이력을 시간 순서로 표시
- 입고 → 이동 → 현재 위치 경로 시각화

## 7. 배치 처리 (시스템)

| 배치 | 주기 | 설명 |
|------|------|------|
| 부진 등급 갱신 | 매일 오전 2시 | 전체 재고의 미이동일수 재계산 및 등급 업데이트 |
| 유효기간 임박 알림 | 매일 오전 9시 | 유효기간 30일 이내 재고 있는 화주사에게 이메일 알림 |
| 365일+ 부진재고 알림 | 매주 월요일 | 365일 이상 미이동 재고 현황 물류사 관리자에게 발송 |
