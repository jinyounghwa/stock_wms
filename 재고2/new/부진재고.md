# 페이지 분석 — 부진재고

## 1. 페이지 개요

| 항목 | 내용 |
|------|------|
| 메뉴 그룹/위치 | 재고관리 > 부진재고 |
| 페이지명 | 부진재고 (꼬리 상품 관리) |
| 목적 | 출고 회전이 낮거나 장기 보관 중인 부진 재고를 SKU 단위로 파악하고, 위치별 점유 현황을 시각화하여 재고 이동·처분·공간 비용 절감 의사결정을 지원. SKU는 여러 위치에 걸쳐 있을 수 있으며, 필터 조건은 SKU 단위로 적용되고 결과 리스트는 위치 단위로 출력 |
| 화면 모드 | **리스트 뷰** ↔ **2D 시각화 뷰 (적치이력)** 토글 전환 |
| 접근 권한 | 화주 (읽기, 자사 데이터 한정), 물류사 관리자, 시스템 관리자 |

> **핵심 설계 원칙**: 미출고 일수·마지막 입고 일수·최소 재고 등의 필터 조건은 SKU 단위로 적용되어야 하며, 결과 리스트는 해당 SKU가 있는 **위치별** 행으로 출력된다.
> 예: SKU A가 3개 위치에 있을 경우 → 리스트에 3행으로 표시

---

## 2. 업무 흐름 (Business Flow)

```
[페이지 진입]
    ↓
필터 설정 (미출고 일수, 마지막 입고 일수, 최소 재고, 시즌 등)
    ↓
[검색] 클릭 → SKU 단위 필터링 → 위치별 결과 리스트 표시
    ↓
결과 검토
  ├── 엑셀 다운로드 → 보고/분석용
  └── 각 상품 행 → [시각화 버튼] 클릭
          ↓
      2D 적치이력 뷰
      (화주사 / 상품명 / 위치 / 재고수량)
      어느 위치에 얼마나 채워져 있는지 2D 카드 표현
      ※ 원아이 개발 진행 중 (Full 시각화)
    ↓
의사결정: 위치 이동 지시 / 재고 처분 / 반품 요청
```

## 3. 필터 영역

| 필드명 | 타입 | 필수 | 적용 단위 | 설명 |
|--------|------|------|----------|------|
| 창고(센터) 선택 | Select | ✅ | 위치 | 조회 대상 물류 센터 |
| 화주사 | Select | - | SKU | 특정 화주사 필터 |
| 구역(존) | Select (다중) | - | 위치 | 입고존 / 보관존 / 출고존 |
| 미출고 일수 | Number | - | SKU | 마지막 출고 이후 N일 이상 경과한 SKU만 표시 (예: 30) |
| 마지막 입고 일수 | Number | - | SKU | 마지막 입고 이후 N일 이상 경과한 SKU만 표시 (예: 30) |
| 최소 재고 | Number | - | SKU | 위치별 최소 재고 N개 이상인 SKU만 표시 (예: 1) |
| 시즌 | Select (다중) | - | SKU | SS(봄/여름) / FW(가을/겨울) / 연중 등 |
| 브랜드 | Select (다중) | - | SKU | 브랜드 필터 |

**[검색] [초기화] 버튼**

> 필터 조건은 SKU 단위로 적용 → 조건을 만족하는 SKU의 모든 위치 행이 결과에 포함됨

## 5. 2D 시각화 뷰 (적치이력)

> **상품별 적치이력 팝업 또는 전환 화면**. 각 상품 행의 [시각화 버튼] 클릭 시 진입.
>
> ※ 풀 시각화(Full 2D) 기능은 **원아이 개발 진행 중** (현재는 간단 버전 2D 제공)

### 5-1. 적치이력 뷰 구성

**헤더 정보 (4가지만 표시)**

| 항목 | 설명 |
|------|------|
| 화주사 | 화주사명 |
| 상품명 | 상품 풀네임 + 옵션 |
| 위치 | 해당 SKU가 적치된 위치 목록 |
| 재고 수량 | 각 위치별 재고 수량 |

**2D 카드 그리드**

해당 SKU가 적치된 위치들을 카드 형태로 그리드 표시:

```
┌──────────────────────────────────────────────┐
│ 상품명: (아동)♥블랙티오비바팬츠[BAM276]       │
│ 옵션: 아이보리 / 3.8(M)   화주사: 언낸모드     │
├──────────────┬──────────────┬────────────────┤
│  재고: 1개  │   재고: 2개  │   재고: 3개   │
│             │              │                │
│ 📍 4E-AX   │  📍 4E-AY   │  📍 4E-AW     │
│   -06-4    │   -06-4     │   -09-4        │
└──────────────┴──────────────┴────────────────┘
```

- 카드마다: 재고 수량 + 위치 코드
- 카드 배경: 재고 수량에 비례한 색상 강도
- **"원아이 개발 진행 중"** 워터마크 또는 배너 표시

**버튼**

| 버튼 | 동작 |
|------|------|
| 리스트로 돌아가기 | 리스트 뷰로 복귀 |
| 엑셀 다운로드 (이 상품) | 해당 SKU 위치별 재고 수량 엑셀 다운로드 |

## 7. API/이벤트

| API | Method | 설명 |
|-----|--------|------|
| `/api/stock/slow-moving/list` | GET | 부진재고 리스트 조회 (SKU 단위 필터 + 위치별 결과) |
| `/api/stock/slow-moving/summary` | GET | 검색 결과 요약 집계 |
| `/api/stock/slow-moving/{skuId}/location-grid` | GET | 특정 SKU의 위치별 적치 현황 (2D 시각화 데이터) |
| `/api/stock/slow-moving/excel-download` | GET | 리스트 엑셀 다운로드 |

## 9. 예외 처리 및 알림

| 상황 | 처리 방식 |
|------|----------|
| 조회 결과 없음 | "조회 조건에 해당하는 부진재고가 없습니다." 안내 |
| 미출고 일수와 마지막 입고 일수 모두 0 입력 | "조건 없이 조회하면 전체 재고가 표시됩니다. 계속하시겠습니까?" 확인 팝업 |
| 엑셀 다운로드 10,000건 초과 | 비동기 처리 후 완료 시 인앱 알림 + 다운로드 링크 |
| 2D 시각화 로드 실패 | "시각화 데이터를 불러오는 중 오류가 발생했습니다. 잠시 후 다시 시도해 주세요." + 리스트 뷰 유지 |
| SKU 최근 출고일 없음 (입고 후 단 한 번도 출고 없음) | 최근 출고일 컬럼 "없음" 표시. 미출고 일수는 입고일로부터 계산 |
