# 페이지 분석 — 위치별 재고 현황

## 1. 페이지 개요

| 항목 | 내용 |
|------|------|
| 메뉴 그룹/위치 | 재고관리 > 위치별 재고 현황 |
| 페이지명 | 위치별 재고 현황 |
| 목적 | 창고 내 각 로케이션(위치)별 재고 점유 현황을 리스트 뷰와 2D 시각화 뷰 두 가지 모드로 제공하여, 위치별 적재 비율·가용 재고·미출고 현황을 한눈에 파악하고 공간 운영 의사결정을 지원 |
| 화면 모드 | **리스트 뷰** ↔ **2D 시각화 뷰** 토글 전환 |
| 접근 권한 | 화주 (읽기, 자사 데이터 한정), 물류사 작업자, 물류사 관리자, 시스템 관리자 |

---

## 2. 업무 흐름 (Business Flow)

```
[페이지 진입]
    ↓
초기 화면: 전체 위치 리스트 + 전체 집계 차트/대시보드
    ↓
필터 설정 (화주, 센터, 구역, 위치 등)
    ↓
┌──────────────┬──────────────────┐
│  리스트 뷰   │   2D 시각화 뷰   │
│ 위치별 행 조회│ 위치 카드 그리드  │
│ 행 선택 → 대 │ 센터별 아코디언  │
│ 시보드 업데이 │ 수량 기반 색상   │
│ 트           │ 표현             │
└──────────────┴──────────────────┘
    ↓
특정 위치 선택 → 해당 위치 상세 (적재 상품 목록, 점유율)
    ↓
엑셀 다운로드 / 재고조사 이동
```

## 3. 공통 필터 영역

| 필드명 | 타입 | 설명 |
|--------|------|------|
| 화주사 | Select | 전체 / 특정 화주사 선택 |
| 센터 | Select | 물류 센터 선택 |
| 구역(존) | Select (다중) | 입고존 / 보관존 / 출고존 등 |
| 위치 | Input | 위치 코드 텍스트 검색 (예: A-01-1) |
| 재고 수량 | NumberRange | 최소~최대 재고 수량 필터 |
| 예약 재고 수량 | NumberRange | 최소~최대 예약 재고 필터 |
| 점유율(%) | NumberRange | 최소~최대 적재 점유율 필터 |

**[검색] [초기화] 버튼**
**[리스트 뷰 ≡] [2D 시각화 ⊞] 토글 버튼** (우측 상단)

## 4. 리스트 뷰

### 4-1. 초기 화면 — 전체 집계 대시보드

리스트 행을 선택하지 않은 초기 상태에서 상단 대시보드에 **전체 현황 차트** 표시:

| 대시보드 항목 | 설명 |
|-------------|------|
| 총 재고 수 | 전체 위치의 재고 합계 |
| 미출고(예약) 재고 수 | 전체 예약 재고 합계 |
| 위치 수 | 조회된 전체 위치 건수 |
| 점유율 분포 차트 | 위치별 점유율 분포 막대/도넛 차트 (전체 기준) |
| 센터별 재고 비율 차트 | 센터별 재고 수량 비율 |

---

### 4-2. 행 선택 후 — 선택 항목 대시보드

리스트에서 체크박스로 1개 이상 위치 선택 시 대시보드 동적 업데이트:

| 대시보드 항목 | 설명 |
|-------------|------|
| 합계 재고 | 선택된 위치들의 재고 수량 합계 |
| 평균 점유율(%) | 선택 위치들의 평균 적재 점유율 |
| 가용 재고 대비 예약 재고 | 가용재고 합계 vs 예약재고 합계 (막대 또는 비율 표시) |
| 선택 위치 수 | 선택된 위치 건수 |

---

### 4-3. 리스트 컬럼

| 컬럼명 | 설명 |
|--------|------|
| 체크박스 | 다중 선택 (대시보드 업데이트 연동) |
| > (펼치기) | 클릭 시 해당 위치 내 상품 목록 인라인 펼침 |
| 센터 | 물류 센터명 (예: A센터) |
| 구역(존) | 존 구분 태그 (입고존 / 보관존 / 출고존 / 기타) |
| 위치 코드 | 위치 식별 코드 (예: A-01-2, 4E-A-01-1) |
| 재고 수량 | 해당 위치의 총 재고 수량 |
| 가용 재고 | 출고 가능한 가용 재고 수량 |
| 예약 재고 | 주문 연계 예약된 재고 수량 |
| 점유율(%) | 해당 위치의 적재 점유율 (재고/최대용량 × 100) |
| 상품 종수 | 해당 위치에 보관 중인 SKU 종류 수 |

---

### 4-4. 위치 펼치기 — 상품 목록 인라인 표시

| 컬럼명 | 설명 |
|--------|------|
| 상품 이미지 | 썸네일 |
| 화주사 | 화주사명 |
| 상품명 / 옵션 | 상품명 + 색상/사이즈 |
| 바코드 | EAN-13 |
| 재고 수량 | 해당 위치의 해당 SKU 수량 |
| 가용 재고 | |
| 예약 재고 | |

---

### 4-5. 액션 버튼

| 버튼 | 권한 | 동작 |
|------|------|------|
| 엑셀 다운로드 | 전체 | 리스트 뷰 현재 조회 결과 엑셀 다운로드 |
| 재고조사 이동 | 작업자 이상 | 선택 위치의 재고조사 탭으로 이동 |

## 5. 2D 시각화 뷰

> 리스트 뷰와 동일한 필터 조건 유지. 뷰 전환 시 검색 조건 그대로 유지.

### 5-1. 화면 상단 — 집계 카드

| 카드 항목 | 설명 |
|----------|------|
| 총 재고 | 전체(또는 필터 기준) 재고 합계 수량 |
| 미발송(예약) | 예약 재고 합계 수량 (주황색 강조) |
| 위치 수 | 조회된 위치 건수 |

---

### 5-2. 정렬 버튼

| 버튼 | 설명 |
|------|------|
| 기본 | 위치 코드 오름차순 |
| 수량 높은순 | 재고 수량 내림차순 |
| 수량 낮은순 | 재고 수량 오름차순 |

---

### 5-3. 센터별 아코디언

각 센터를 아코디언으로 구분:

```
[A센터]  재고: 137,612  미발송: 15,403  [N개 더보기]
  ┌─────────┐ ┌─────────┐ ┌─────────┐ ┌─────────┐
  │  재고량  │ │  재고량  │ │  재고량  │ │  재고량  │
  │    0    │ │    0    │ │    0    │ │    1    │
  │미발송    │ │미발송    │ │미발송    │ │미발송    │
  │  없음   │ │  없음   │ │  없음   │ │  없음   │
  │a7g90122 │ │a7g87954 │ │A7JE0001 │ │A7V00001 │
  └─────────┘ └─────────┘ └─────────┘ └─────────┘
```

**위치 카드 구성**

| 항목 | 설명 |
|------|------|
| 재고량 레이블 | "재고량" 라벨 |
| 재고 수량 | 해당 위치 재고 수 (숫자) |
| 미발송 레이블 | "미발송" 라벨 |
| 미발송 수량 | 예약 재고 수 |
| 위치 코드 | 하단 위치 식별자 |
| 카드 배경색 | 점유율/재고 수량에 따른 색상 강도 (낮음: 흰색, 높음: 진한 파란색) |
| 미발송 강조 | 미발송 수량 > 0 시 주황색 표시 |

**카드 클릭** → 해당 위치 상세 팝업 (보관 상품 목록, 가용/예약/점유율 상세)

**N개 더보기** → 해당 센터의 나머지 위치 카드 추가 표시

---

### 5-4. 2D 뷰 색상 체계

| 점유율 구간 | 배경색 | 의미 |
|-----------|--------|------|
| 0% (재고 없음) | White | 빈 위치 |
| 1~30% | Light Blue | 여유 |
| 31~60% | Medium Blue | 보통 |
| 61~90% | Deep Blue | 충분 |
| 91~100% | Dark Blue | 만재 |
| 미발송 있음 | Orange 테두리 강조 | 출고 대기 재고 포함 |

## 7. API/이벤트

| API | Method | 설명 |
|-----|--------|------|
| `/api/stock/location/list` | GET | 위치별 재고 리스트 조회 (필터 파라미터) |
| `/api/stock/location/summary` | GET | 집계 대시보드 데이터 |
| `/api/stock/location/{locationId}/products` | GET | 특정 위치 내 상품 목록 |
| `/api/stock/location/2d-grid` | GET | 2D 시각화용 그리드 데이터 (센터별 위치 카드) |
| `/api/stock/location/excel-download` | GET | 리스트 엑셀 다운로드 |

## 9. 예외 처리 및 알림

| 상황 | 처리 방식 |
|------|----------|
| 조회 결과 없음 | "조회 조건에 해당하는 위치가 없습니다." 안내 |
| 최대 용량 미설정 위치 | 점유율 계산 불가. "-" 표시, 툴팁으로 "위치 최대 용량이 미설정된 위치입니다." 안내 |
| 엑셀 다운로드 5,000건 초과 | 비동기 처리 후 완료 시 인앱 알림 + 다운로드 링크 |
| 실시간 재고 조회 오류 | "재고 데이터를 불러오는 중 오류가 발생했습니다. 잠시 후 다시 시도해 주세요." |
