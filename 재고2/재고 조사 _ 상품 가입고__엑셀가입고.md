# 페이지 분석 — 엑셀가입고

## 1. 페이지 개요

| 항목 | 내용 |
|------|------|
| 메뉴 그룹/위치 | 재고 조사 > 상품 가입고 |
| 페이지명 | 엑셀가입고 |
| 식별 코드 | 미기재 |
| 목적 | 엑셀(XLSX) 또는 CSV 파일을 업로드하여 다수의 SKU 가입고 수량을 일괄 등록하는 메뉴. 시스템에서 내려받은 전산재고 목록에 실사 수량을 채워 넣은 파일을 올리는 방식으로 사용 |
| 접근 권한 | 물류사 작업자, 물류사 관리자, 시스템 관리자 |

> **패션 WMS 적용 시나리오**: 시즌 말 전수 재고조사 시, 담당자가 시스템에서 전산재고 목록을 XLSX로 내려받아 사무실에서 실사 수량을 입력한 후, 이 메뉴로 파일을 업로드하는 방식으로 대량 가입고를 처리한다.

---

## 2. 업무 흐름 (Business Flow)

```
[전산재고 목록 다운로드] (엑셀 서식 다운로드)
    ↓
현장 실사 후 엑셀 파일에 실사 수량 직접 입력
    ↓
[엑셀가입고] 메뉴 진입
    ↓
화주 · 창고 · 조사차수 선택
    ↓
파일 업로드 (드래그앤드롭 또는 파일 선택)
    ↓
시스템 파일 유효성 검사
  ├── 성공 → 미리보기 표시
  └── 실패 → 오류 목록 표시
    ↓
미리보기에서 내용 확인 후 [등록 실행]
    ↓
가입고 등록 완료 → R-STK-603 (가입고 목록) 반영
    ↓
업로드 결과 리포트 표시 (성공/실패 건수)
```

## 3. 화면 구성

### 3-1. 헤더 설정 영역

| 필드명 | 타입 | 필수 | 설명 |
|--------|------|------|------|
| 화주 | Select | ✅ | 가입고 대상 화주 |
| 창고 | Select | ✅ | 가입고 대상 창고 |
| 조사 차수 | Select | ✅ | 현재 진행 중인 재고조사 회차 |

---

### 3-2. 파일 업로드 영역

| 요소 | 설명 |
|------|------|
| 업로드 영역 | 드래그앤드롭 또는 "파일 선택" 버튼. 허용 형식: .xlsx, .xls, .csv |
| 파일 크기 제한 | 최대 10MB |
| 행 수 제한 | 최대 10,000행 |
| 서식 다운로드 버튼 | 가입고 엑셀 업로드용 서식 파일 다운로드 |

---

### 3-3. 업로드 엑셀 서식 컬럼 정의

| 컬럼명 | 필수 | 설명 |
|--------|------|------|
| 상품코드 | ✅ (바코드 없는 경우) | 내부 SKU 코드 |
| 바코드 | ✅ (상품코드 없는 경우) | EAN-13 바코드. 상품코드와 둘 중 하나 필수 |
| 로케이션 | - | 보관 위치 코드. 미입력 시 창고 전체로 처리 |
| 가입고수량 | ✅ | 실사 수량. 1 이상 정수 |
| 비고 | - | 특이사항 메모 (최대 100자) |

> **서식 추가 제공 (읽기전용 참조 컬럼)**: 상품명, 스타일코드, 색상, 사이즈, 전산재고 — 담당자가 실사 수량을 채울 때 참고용으로만 사용. 업로드 시 해당 컬럼 값은 무시됨.

---

### 3-4. 파일 업로드 후 미리보기

| 컬럼명 | 설명 |
|--------|------|
| 행 번호 | 엑셀 행 번호 |
| 상품코드 | 업로드된 상품코드 |
| 바코드 | 업로드된 바코드 |
| 매칭 상품명 | 시스템에서 조회된 상품명 |
| 스타일코드 | 조회된 스타일 코드 |
| 색상 | 조회된 색상 |
| 사이즈 | 조회된 사이즈 |
| 로케이션 | 업로드된 로케이션 |
| 가입고수량 | 업로드된 수량 |
| 유효성 | 정상 (Green) / 오류 (Red, 사유 표시) |

> **오류 건 처리**: 유효성 오류가 있는 행은 빨간색으로 표시. 오류 행 제외하고 정상 행만 등록 가능 (옵션 선택).

---

### 3-5. 업로드 결과 화면

| 항목 | 설명 |
|------|------|
| 총 업로드 행수 | 파일 내 전체 데이터 행수 |
| 성공 건수 | 가입고 등록 성공 건수 |
| 실패 건수 | 유효성 오류 또는 시스템 오류로 실패한 건수 |
| 실패 원인 목록 | 실패 행번호, 상품코드/바코드, 실패 사유 |
| 실패 건 엑셀 다운로드 | 실패 건만 별도 엑셀 다운로드 버튼 |

### 3-6. 액션 버튼

| 버튼명 | 권한 | 동작 |
|--------|------|------|
| 서식 다운로드 | 전체 | 가입고 엑셀 업로드 서식 다운로드 |

## 4. 유효성 검사 규칙

| 오류 유형 | 설명 |
|----------|------|
| 필수 컬럼 누락 | 상품코드와 바코드 모두 비어있는 행 |
| 미등록 상품 | 상품코드 또는 바코드로 시스템에서 상품을 찾을 수 없는 경우 |
| 수량 오류 | 가입고수량이 0 이하이거나 정수가 아닌 경우 |
| 미등록 로케이션 | 입력된 로케이션 코드가 시스템에 존재하지 않는 경우 |
| 중복 행 | 동일 차수 내 동일 SKU·로케이션 조합 중복 시 경고 (합산 처리 또는 오류 처리 선택) |
