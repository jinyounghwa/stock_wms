# 페이지 분석 — 차수별엑셀가입고 리스트

## 1. 페이지 개요

| 항목 | 내용 |
|------|------|
| 메뉴 그룹/위치 | 재고 조사 > 상품 가입고 |
| 페이지명 | 차수별엑셀가입고 리스트 |
| 식별 코드 | 미기재 |
| 목적 | 엑셀가입고 메뉴를 통해 업로드된 엑셀 파일 이력을 일자/차수별로 조회하고 관리하는 메뉴. 업로드 결과(성공/실패 건수) 확인, 원본 파일 재다운로드, 업로드 취소 기능을 제공 |
| 접근 권한 | 물류사 작업자(읽기), 물류사 관리자(전체), 시스템 관리자(전체) |

---

## 2. 업무 흐름 (Business Flow)

```
[엑셀가입고] 에서 파일 업로드 완료
    ↓
업로드 이력이 [차수별엑셀가입고 리스트]에 자동 기록
    ↓
목록에서 업로드 결과(성공/실패 건수) 확인
    ↓
실패 건 있는 경우
    ├── 실패 건 엑셀 다운로드 → 오류 수정 후 재업로드
    └── 업로드 취소 (전체 롤백) → 재업로드
```

### 3-2. 목록 컬럼 (업로드 배치 단위)

| 컬럼명 | 설명 |
|--------|------|
| 업로드번호 | 엑셀 업로드 배치 식별자 (예: EXUP-20240315-001) |
| 업로드일시 | 파일 업로드 실행 일시 |
| 조사 차수 | 연계된 재고조사 회차 |
| 화주사 | 화주사명 |
| 창고 | 창고명 |
| 업로드 파일명 | 원본 업로드 파일명 |
| 총 행수 | 업로드 파일의 데이터 행수 (헤더 제외) |
| 성공 건수 | 가입고 등록 성공 건수 (Green 표시) |
| 실패 건수 | 유효성 오류 등으로 실패한 건수 (Red 표시, 0이면 표시 없음) |
| 처리율 | 성공 / 총 행수 × 100 (%) |
| 상태 | 처리중 / 완료 / 실패포함 / 취소됨 (뱃지) |
| 업로드자 | 파일을 업로드한 작업자명 |
| 처리완료일시 | 업로드 처리가 완료된 일시 |

| 컬럼명 | 설명 |
|--------|------|
| 행 번호 | 원본 엑셀 파일의 행 번호 |
| 상품코드 | 업로드된 상품코드 |
| 바코드 | 업로드된 바코드 |
| 상품명 | 조회된 상품명 |
| 스타일코드 | 스타일 코드 |
| 색상 | 색상명 |
| 사이즈 | 사이즈 |
| 로케이션 | 로케이션 코드 |
| 가입고수량 | 업로드된 수량 |
| 처리결과 | 성공 (Green) / 실패 (Red) |
| 실패사유 | 실패 원인 설명 (성공 시 '-') |

---

### 3-4. 액션 버튼

| 버튼명 | 권한 | 동작 |
|--------|------|------|
| 원본 파일 다운로드 | 전체 | 업로드한 원본 엑셀 파일 재다운로드 |
| 실패 건 다운로드 | 전체 | 실패한 행만 별도 엑셀 다운로드 (수정 후 재업로드 용) |
| 업로드 취소 | 관리자 이상 | 해당 업로드 배치 전체 롤백 (가입고 등록 취소). 이미 실입고 처리된 건 있으면 취소 불가 |
| 엑셀 다운로드 (목록) | 전체 | 조회된 업로드 배치 목록 엑셀 다운로드 |

## 4. 상태값 정의

| 상태 | 설명 | 색상 |
|------|------|------|
| 처리중 | 업로드 파일 처리 진행 중 (대용량 시 비동기) | Blue |
| 완료 | 모든 행 성공 처리 완료 | Green |
| 실패포함 | 일부 행 실패 있으나 성공 행은 등록 완료 | Orange |
| 전체실패 | 전체 행 실패 (등록된 가입고 없음) | Red |
| 취소됨 | 관리자에 의해 업로드 배치 취소됨 | Gray |

## 6. API/이벤트

| API | Method | 설명 |

| `/api/stock/pre-receipt/excel-upload-history/list` | GET | 엑셀 업로드 배치 목록 조회 |
| `/api/stock/pre-receipt/excel-upload-history/{id}/detail` | GET | 특정 업로드 배치 재고별 상세 결과 |
| `/api/stock/pre-receipt/excel-upload-history/{id}/original` | GET | 원본 파일 다운로드 |
| `/api/stock/pre-receipt/excel-upload-history/{id}/failures` | GET | 실패 건 엑셀 다운로드 |
| `/api/stock/pre-receipt/excel-upload-history/{id}/cancel` | POST | 업로드 배치 취소 (롤백) |

## 8. 예외 처리 및 알림

| 상황 | 처리 방식 |
|------|----------|
| 이미 실입고 처리된 건 포함된 배치 취소 시도 | "해당 업로드 배치 중 이미 실입고 처리된 항목이 있어 취소할 수 없습니다. 가입고 목록에서 개별 처리하세요." 에러 |
| 대용량 파일 업로드 처리 중 | 처리중 상태로 표시. 완료 시 인앱 알림 발송 |
| 원본 파일 보관 만료 (기본 90일) | 파일 다운로드 버튼 비활성화 + "보관 기간이 만료된 파일입니다." 안내 |
