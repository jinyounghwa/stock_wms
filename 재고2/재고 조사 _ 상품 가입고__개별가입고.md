# 페이지 분석 — 개별가입고

## 1. 페이지 개요

| 항목 | 내용 |
|------|------|
| 메뉴 그룹/위치 | 재고 조사 > 상품 가입고 |
| 페이지명 | 개별가입고 |
| 식별 코드 | 미기재 |
| 목적 | 재고조사 중 바코드 스캐너나 엑셀 사용이 어려운 환경에서 상품코드/바코드를 직접 입력하여 1건씩 가입고 수량을 등록하는 메뉴. 소량 품목 또는 특이 상품 처리에 활용 |
| 접근 권한 | 물류사 작업자, 물류사 관리자, 시스템 관리자 |

---

## 2. 업무 흐름 (Business Flow)

```
[재고조사 진행 중]
    ↓
[개별가입고] 진입
    ↓
화주사 · 창고 · 조사차수 선택
    ↓
상품코드 또는 바코드 입력 → 상품 정보 자동 조회
    ↓
로케이션 선택 (선택 사항)
    ↓
가입고 수량 입력
    ↓
[추가] 버튼 → 하단 임시 목록에 행 추가
    ↓
추가 품목이 있으면 반복
    ↓
[저장(일괄 등록)] → 가입고 목록(R-STK-603)에 반영
```

---

## 3. 화면 구성

### 3-1. 헤더 입력 영역 (공통 정보)

| 필드명 | 타입 | 필수 | 설명 |
|--------|------|------|------|
| 화주사 | Select | ✅ | 가입고 대상 화주사 |
| 창고 | Select | ✅ | 가입고 대상 창고 |
| 조사 차수 | Select | ✅ | 현재 진행 중인 재고조사 회차 |

---

### 3-2. 상품 입력 영역 (행 단위 반복 입력)

| 필드명 | 타입 | 필수 | 설명 |
|--------|------|------|------|
| 상품코드 또는 바코드 | Input | ✅ | 직접 입력. 입력 후 Tab/Enter 시 상품 정보 자동 조회 |
| 브랜드 | 읽기전용 | - | 상품코드 입력 후 자동 표시 |
| 스타일코드 | 읽기전용 | - | 자동 표시 |
| 색상 | 읽기전용 | - | 자동 표시 |
| 사이즈 | 읽기전용 | - | 자동 표시 |
| 상품명 | 읽기전용 | - | 자동 표시 |
| 전산재고 | 읽기전용 | - | 자동 조회. 현재 전산재고 수량 표시 (참고용) |
| 로케이션 | Select + Input | - | 보관 위치 선택 또는 직접 입력 |
| 가입고 수량 | Number | ✅ | 1 이상 정수 입력 |
| 비고 | Input | - | 특이사항 메모 |

**[추가] 버튼**: 현재 입력 행을 하단 임시 목록에 추가하고, 입력 폼 초기화 (상품코드 필드에 포커스 이동)

---

### 3-3. 임시 목록 (하단 - 저장 전 검토 영역)

| 컬럼명 | 설명 | 수정 |
|--------|------|------|
| No | 행 번호 | ✗ |
| 상품코드 | SKU 코드 | ✗ |
| 바코드 | EAN-13 | ✗ |
| 브랜드 | 브랜드명 | ✗ |
| 스타일코드 | 스타일 코드 | ✗ |
| 색상 | 색상명 | ✗ |
| 사이즈 | 사이즈 | ✗ |
| 상품명 | 상품 풀네임 | ✗ |
| 로케이션 | 로케이션 코드 | ✅ |
| 가입고 수량 | 입력 수량 | ✅ |
| 비고 | 메모 | ✅ |
| 삭제 | 행 삭제 버튼 | - |

> 임시 목록은 브라우저 세션 내에서만 유지. **[저장]** 전까지 서버에 저장되지 않음.

---

### 3-4. 하단 요약

| 항목 | 설명 |
|------|------|
| 총 입력 건수 | 임시 목록의 행 수 |
| 총 입력 수량 | 가입고 수량 합계 |

---

### 3-5. 액션 버튼

| 버튼명 | 권한 | 동작 |
|--------|------|------|
| 추가 | 작업자 이상 | 현재 입력 행을 임시 목록에 추가 |
| 저장 (일괄 등록) | 작업자 이상 | 임시 목록 전체를 가입고로 서버에 저장 |
| 전체 삭제 | 작업자 이상 | 임시 목록 전체 삭제 (확인 팝업) |
| 가입고 목록으로 이동 | 전체 | R-STK-603 (가입고 목록)으로 이동 |

---

## 4. 입력 검증 규칙

| 필드 | 규칙 |
|------|------|
| 상품코드/바코드 | 시스템에 등록된 상품만 허용. 미등록 시 "등록되지 않은 상품입니다." 에러 |
| 가입고 수량 | 1 이상 정수만 허용 |
| 로케이션 | 입력 시 시스템에 존재하는 로케이션 코드만 허용 |
| 동일 SKU 중복 입력 | 같은 차수 내 동일 SKU·로케이션 조합 추가 시 "동일 상품이 이미 목록에 있습니다. 수량을 합산하시겠습니까?" 팝업 |
| 저장 시 임시 목록 빈값 | "등록할 항목이 없습니다." 안내 |

## 8. UX 고려사항

- **키보드 친화적 입력**: 바코드 입력 → Tab → 로케이션 → Tab → 수량 → Enter(=추가) 흐름으로 마우스 없이 연속 입력 가능해야 함
- **상품 미등록 대응**: 미등록 바코드 입력 시 상품 등록 페이지로 바로 이동할 수 있는 링크 제공
- **화면 이탈 방지**: 임시 목록에 미저장 데이터가 있을 때 페이지 이탈 시 "저장되지 않은 데이터가 있습니다." 경고
