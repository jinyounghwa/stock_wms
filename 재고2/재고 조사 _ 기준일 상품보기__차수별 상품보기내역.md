# 페이지 분석 — 차수별 상품보기내역

## 1. 페이지 개요

| 항목 | 내용 |
|------|------|
| 메뉴 그룹/위치 | 재고 조사 > 기준일 상품보기 |
| 페이지명 | 차수별 상품보기내역 |
| 식별 코드 | 미기재 |
| 목적 | 기준일 상품보기(R-STK-607) 메뉴에서 진행된 작업 이력을 조사 차수(회차) 단위로 집계·조회하는 이력 관리 메뉴. 차수별 조회 건수, 실산 완료율, 가입고 연계 현황을 한눈에 파악하고, 특정 차수 클릭 시 SKU별 상세 작업 이력을 드릴다운하여 확인할 수 있다 |
| 접근 권한 | 물류사 작업자(읽기 전용), 물류사 관리자, 시스템 관리자 |

> **활용 시나리오**: 재고조사 종료 후 감사(Audit) 또는 화주 보고 시, 각 조사 차수별로 몇 개 SKU를 조회했고 실산을 완료했으며 가입고와 얼마나 연계됐는지 집계 현황을 확인하는 용도. 또한 특정 차수에서 조회했지만 실산을 완료하지 않은 미처리 항목을 추적하여 후속 작업을 지시할 때 사용한다.

---

## 2. 업무 흐름 (Business Flow)

```
[재고조사 진행 중 또는 완료 후]
    ↓
[기준일 상품보기] (R-STK-607) → [작업내역보기] / [등록상품보기] 탭에서
담당자들이 차수별로 상품 조회 및 실산 입력
    ↓
[차수별 상품보기내역]에서 차수별 집계 결과 확인
    ↓
특정 차수 클릭 → 해당 차수의 SKU별 작업 상세 내역 드릴다운
    ↓
미실산 항목 확인 → 담당자에게 추가 작업 지시
또는
엑셀 다운로드 → 화주 보고 / 내부 감사 자료 활용
```

| 최초 작업일시 | 해당 차수 내 첫 번째 상품조회 일시 |
| 최종 작업일시 | 해당 차수 내 마지막 작업 일시 |

### 3-3. SKU별 상세 내역 (드릴다운 / 하위 패널)

차수 행 클릭 시 하단 또는 별도 패널에 해당 차수의 SKU별 작업 내역 표시:

| 컬럼명 | 설명 |
|--------|------|
| 조회일시 | 상품을 조회한 일시 |
| 조회 담당자 | 조회한 작업자명 |
| 브랜드 | 브랜드명 |
| 상품코드 | SKU 코드 |
| 바코드 | EAN-13 바코드 |
| 스타일코드 | 스타일 코드 |
| 색상 | 색상명 |
| 사이즈 | 사이즈 표기 |
| 상품명 | 상품 풀네임 |
| 로케이션 | 조회 당시 로케이션 |
| 기준일 재고 | 기준일 전산재고 스냅샷 |
| 가입고 수량 | 해당 차수 가입고 누계 |
| 실산수량 | 입력된 실산수량 (미입력 시 '-') |
| 차이수량 | 실산수량 - 기준일 재고 |
| 실산 상태 | 입력완료 (Green) / 미입력 (Orange) / 수정됨 (Blue) |
| 가입고 연계 | 연계된 가입고 건수 및 링크 |
| 재고조정 연계 | 연계된 재고조정 요청 번호 |
| 메모 | 실산 메모 |

| 실산완료 | 조회 SKU 전체 실산수량 입력 완료 | 해당 차수의 모든 조회 상품 실산 완료 | Green |

## 8. 예외 처리 및 알림

| 상황 | 처리 방식 |
|------|----------|
| 조회 결과 없음 | \"조회 조건에 해당하는 작업 내역이 없습니다.\" 안내 |
| 조회 기간 범위 과다 (1년 초과) | \"조회 기간은 최대 1년(365일)까지 설정할 수 있습니다.\" 경고 |
| 대용량 엑셀 다운로드 (10,000건 초과) | 비동기 처리 후 완료 시 인앱 알림 + 다운로드 링크 제공 |
| 특정 차수 데이터 정합성 오류 | 해당 차수에 경고 아이콘 표시 및 시스템 관리자 알림 |
