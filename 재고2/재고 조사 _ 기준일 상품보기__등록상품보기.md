# 페이지 분석 — 등록상품보기

## 1. 페이지 개요

| 항목 | 내용 |
|------|------|
| 메뉴 그룹/위치 | 재고 조사 > 기준일 상품보기 |
| 페이지명 | 등록상품보기 |
| 식별 코드 | 미기재 |
| 목적 | 재고조사 담당자가 바코드번호 또는 상품코드 목록이 담긴 CSV 파일을 업로드하면, 해당 상품들의 기준일 기준 재고 현황(기준일 재고, 가입고 수량, 전산재고 등)을 일괄 조회·확인할 수 있는 메뉴 |
| 접근 권한 | 물류사 작업자(읽기), 물류사 관리자, 시스템 관리자 |

> **활용 시나리오**: 화주사가 자체 시스템에서 관리하는 상품 목록(바코드 리스트)을 제공하면, 물류사 담당자가 이 메뉴에서 해당 상품들의 WMS 기준 재고를 일괄 조회하여 재고 현황을 화주사에게 공유하는 용도로 많이 사용된다.

---

## 2. 업무 흐름 (Business Flow)

```
[기준일 상품보기] (R-STK-607) 에서 [등록상품보기] 탭/메뉴 진입
    ↓
화주사 · 창고 · 기준일 · 조사차수 설정
    ↓
방법 1: CSV 파일 업로드 (바코드 또는 상품코드 컬럼)
방법 2: 직접 입력창에 바코드/상품코드 붙여넣기 (줄바꿈 구분)
    ↓
[조회] → 입력된 상품코드/바코드에 해당하는 기준일 재고 현황 표시
    ↓
결과 확인 → 엑셀 다운로드 or 가입고 입력(R-STK-602)으로 이동
```

## 3. 화면 구성

### 3-1. 검색 조건 (헤더 영역)

| 필드명 | 타입 | 필수 | 설명 |
|--------|------|------|------|
| 화주사 | Select | ✅ | 조회 대상 화주사 |
| 창고 | Select | ✅ | 조회 대상 창고 |
| 기준일 | Date | ✅ | 재고 스냅샷 기준일. 기본값: 최신 차수의 기준일 |
| 조사 차수 | Select | - | 특정 재고조사 회차 선택 |

---

### 3-2. 상품 목록 입력 방식

**방식 A: CSV 파일 업로드**

| 항목 | 설명 |
|------|------|
| 허용 형식 | .csv, .txt, .xlsx |
| 필수 컬럼 | 바코드 또는 상품코드 (둘 중 하나 이상 포함) |
| 행 수 제한 | 최대 5,000행 |
| 서식 다운로드 | 업로드용 CSV 서식 다운로드 버튼 제공 |

**방식 B: 직접 입력**

- 텍스트 영역에 바코드 또는 상품코드를 줄바꿈 또는 콤마로 구분하여 붙여넣기
- 최대 500건

---

### 3-3. 조회 결과 목록 컬럼

| 컬럼명 | 설명 |
|--------|------|
| 입력 바코드/상품코드 | 조회에 사용된 원본 입력값 |
| 매칭 여부 | 매칭됨 (Green) / 미등록 (Red) |
| 화주사 | 화주사명 |
| 창고 | 창고명 |
| 브랜드 | 패션 브랜드명 |
| 상품코드 | 내부 재고 코드 |
| 바코드 | EAN-13 바코드 |
| 스타일코드 | 스타일 코드 |
| 색상 | 색상명 |
| 사이즈 | 사이즈 표기 |
| 상품명 | 상품 풀네임 |
| 기준일 재고 | 기준일 당시 전산재고 스냅샷 수량 |
| 가입고 수량 | 해당 차수에서의 가입고 누적 수량 |
| 현재고 | 현재 전산재고 수량 |
| 로케이션 | 보관 중인 로케이션 (복수 로케이션 시 콤마 구분 또는 펼치기) |
| 조사 상태 | 미조사 / 조사중 / 조사완료 |

---

### 3-4. 요약 정보 (목록 상단)

| 항목 | 설명 |
|------|------|
| 입력 건수 | 업로드/입력된 바코드·상품코드 총 건수 |
| 매칭 건수 | 시스템에서 상품 정보 조회 성공 건수 |
| 미등록 건수 | 시스템에 등록되지 않은 바코드·상품코드 건수 |

---

### 3-5. 액션 버튼

| 버튼명 | 권한 | 동작 |
|--------|------|------|
| 서식 다운로드 | 전체 | 조회용 CSV 서식 다운로드 |
| 조회 | 전체 | 입력된 상품 목록 기준 재고 조회 |
| 엑셀 다운로드 | 전체 | 조회 결과 엑셀 다운로드 |
| 가입고 입력 | 작업자 이상 | 조회된 상품으로 R-STK-602 가입고 등록 이동 |

---

## 4. 입력 CSV 서식 정의

| 컬럼 | 필수 | 설명 |
|------|------|------|
| 바코드 | ✅ (상품코드 없는 경우) | EAN-13 바코드 |
| 상품코드 | ✅ (바코드 없는 경우) | 내부 재고 코드 |

단, 바코드와 상품코드 둘 다 있는 경우 바코드 우선 적용.
